{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Preview Prototypes Prototypes can be previewed on any mobile device connected to the same Wi-Fi network. You can preview your projects on WebKit-based browsers or on one of the native iOS and Android apps. --- ##### Get Started To start previewing your prototypes, click the **"Mirror"** icon in the toolbar. You can open the link presented in the popover menu on your mobile device. The share button can be used to quickly send yourself or others the link via message or email.
  • Connect via WiFi

    Make sure your Mac and mobile devices are on same Wi-Fi network.

  • Preview on WebKit Browsers

    Preview your projects on any WebKit-based browser.

  • Preview on a Native App

    For an optimal, full-screen experience, see the native previewing apps.

##### How it works Framer Studio runs a built-in webserver containing all of your open projects. The address is your Network **IP address**. On save, Framer Studio automatically refreshes your project for live-previewing. The web server doesn't rely on Bonjour, so it works on all networks, including corporate managed networks. ---
