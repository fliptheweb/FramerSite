{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Sketch Tips ##### Position on Canvas The x and y position of your layers are turned into x and y properties in Framer. This means that layers with negative coordinates are positioned off-canvas in Framer. If your layers are contained within a parent group, ensure that its x and y position is set to zero. This doesn’t apply when designing within Artboards.
##### Hidden Layers Layers that you’ve hidden are also imported, but with visibility set to "false" by default. The opacity property is left unchanged. This feature is currently only available in Sketch Beta.
##### Artboards Only the first artboard will be visible by default. All other artboards are hidden, and positioned on top of eachother. The x and y properties of imported artboards are always set to zero.
