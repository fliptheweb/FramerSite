{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Sketch Tips ##### Position on Canvas The x and y positions of your layers on the canvas are imported as well. This causes layers with negative coordinates to appear off-canvas in Framer. Unless if you're designing within artboards, make sure that the x and y position of your parent layer group are both set to 0.
##### Hidden Layers Hidden layers are also imported, but with visibility set to false by default. The opacity property is left unchanged. Only in Sketch Beta for now.
##### Artboards Only the first artboard will be visible by default, and every other artboard will be imported with the x and y positions set to 0 by default.
