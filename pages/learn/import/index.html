{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Import Designs Framer Studio lets you import your static mockups from Photoshop or Sketch with a single click. All of your layers are available and you can continuously update your designs while bringing them to life. --- ##### Get Started Start by clicking the **"Import"** icon in the toolbar.  You'll be presented with the Import sheet, which informs you about the state of the applications. Note that you can only import from documents that you’ve opened and previously saved.
### Set-up Let’s have a look at how to set-up your documents before importing. Make sure that your layers are placed in groups and have unique names. Grouped layers can be targeted by name in Framer. Ungrouped layers are either flattened into a group or become part of the background of their parent group.
##### Targeting layers After import, an import statemt gets inserted in the code. By referencing it, you're able to target all of the imported layers within Framer. 
# This imports all the layers of "design" into designLayers
designLayers = Framer.Importer.load "imported/design"
Note that the default variable name can be changed to anything.
Let’s simply rename it to “import”. Now, if you'd like to target a layer group named “layerA” in Photoshop or Sketch, you can simply write: 
import = Framer.Importer.load "imported/design"
import.layerA.opacity = 0.5
##### Targeting multiple layers You can use a for-loop to target multiple layers. You can either loop over all of your imported layers, or target the layers of a specific layer group.
