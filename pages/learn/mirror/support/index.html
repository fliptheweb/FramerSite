{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Support #### Gotchas - If you switch networks or change your network settings, you may need to restart Framer Studio. - If your network is really slow you can use this tethering trick to connect via USB instead of WiFi. - If Framer Studio can't use Port 8000 it will try to use the next available Port (8001, 8002 etc). - The minimal-ui viewport tag is deprecated in iOS8 and throws an error on desktop so it's not enabled by default. - The apple-mobile-web-app-capable tag has a bug where consequent launches can freeze your phone so it's not enabled by default. It seems to be fixed in iOS8. --- #### Troubleshooting **If your device can't find the server on the network:** - Double check if both devices are on the same network - Try to use you local dns name rather than the IP address. Look it up in System Preferences > Sharing. It should look something like http://my-computer.local:8000/MyProject.framer - Check if you network firewall is blocking traffic on ports 8000-9000 {% endfilter %} {% endfilter %} {% endblock body %}
