Transparent Videos

With After Effects, it's possible to export fully transparant videos. This can be very useful for prototyping purposes. Note that animating transparant videos may showcase some jagged edges due to anti-aliasing. If possible, exporting with a background is recommended.

  • Toggle Transparency grid on your composition
  • Add your project to the Render Queue
  • In the Output Module, select the RGB + Alpha channel
  • In the Format Options, use the Animation Codec
--- #### Notes ##### Chrome doesn't support .MOV Framer and Safari support Lossless .MOV straight from AE. ##### Web-compatible Prototypes To create web-compatible videos, convert them to MP4 or export using AME, and use .H264 as the Codec and .MP4 as file format ---
