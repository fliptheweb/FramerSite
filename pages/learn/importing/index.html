{% filter pygmentify %} {% filter markdown %} #### Overview The importer lets you import Sketch or Photoshop documents directly into Framer. All of your layer groups will be available within Framer. Your layer hierarchy is respected and all flattened layers are exported too. Subsequent imports from the same file will work as long as you don't rename layers in Photoshop or Sketch that you've already targeted within Framer. This allows for rapid iteration. --- #### Principles - Every *layer group* in Sketch/Photoshop will become a *layer* in Framer. - Layer hierarchy is respected, eg: a subgroup will become a subLayer. - Masked layer groups will become layers with clip enabled. - Imported layers are accessible by the importedLayers object: “importedLayers.myLayer”. - Group names should be unique. --- #### Gotchas - Only active documents will be imported. - Before importing a document, it needs to be saved at least once. - Layers are imported as a flat list. So, to access a layer named “ChildLayer” you write “importedLayers.ChildLayer” instead of “importedLayers.ParentLayer.ChildLayer”. - When a layer group contains both other groups and other layers - the subgroups will be placed on top of the sublayers. - To create a scrolling layer, ensure it contains a group that exceeds its size. - In Sketch, all artboards are imported, but only the first one will be visible by default. - In Sketch, you can only use one mask per layer group. - In Sketch, make sure you have the elements you like to import don't have negative coordinates as they'll be hidden within Framer. --- #### Compatibility:
Application Compatibility
Sketch 3 Compatible, not recommended
Sketch 3.1 Beta Fully compatible
Photoshop CS6 Fully compatible
Photoshop CC Fully compatible
Photoshop CC 2014 Fully compatible
