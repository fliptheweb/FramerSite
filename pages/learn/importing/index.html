Importing

Import files from Sketch, Photoshop or After Effects and turn them into interactive prototypes.
{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### Overview The importer lets you import Sketch or Photoshop documents directly into Framer. All of your layer groups will be available. Your layer hierarchy is respected and all flattened layers are exported too. Subsequent imports from the same file will work as long as you don't rename layers that you've already targeted in Framer. This allows for rapid iteration. Only active & saved documents will be imported. --- ### Principles ##### Layers - Layer hierarchy is respected, eg: a subgroup will become a subLayer - Layers are imported as a flat list. To target **“ChildLayer”** you write **"imported.ChildLayer”** instead of **"imported.ParentLayer.ChildLayer”** - Masked layer groups will become layers with clip enabled ##### Photoshop & Sketch - Every *layer group* will become a *layer* in Framer - Layer group names should be unique ##### Sketch - All artboards are imported, but only the first one will be visible by default - Elements with negative coordinates will not be visible in Framer --- #### Compatibility:
Application Compatibility
Sketch 3 Compatible, not recommended
Sketch 3.1 Beta Fully compatible
Photoshop CS6 Fully compatible
Photoshop CC Fully compatible
Photoshop CC 2014 Fully compatible
