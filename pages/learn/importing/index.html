Importing

Import files from Sketch, Photoshop or After Effects and turn them into interactive prototypes.
{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### Overview The importer lets you import Sketch or Photoshop documents directly into Framer. All of your layer groups will be available. Your layer hierarchy is respected and all flattened layers are exported too. Subsequent imports from the same file will work as long as you don't rename layers that you've already targeted within Framer. This allows for rapid iteration. Only active & saved documents will be imported. --- #### Principles - Every *layer group* in Sketch/Photoshop will become a *layer* in Framer. - Layer hierarchy is respected, eg: a subgroup will become a subLayer. - Masked layer groups will become layers with clip enabled. - Group names should be unique. --- #### Gotchas - Layers are imported as a flat list. So, to access a layer named “ChildLayer” you write “sketchLayers.ChildLayer” instead of “sketchLayers.ParentLayer.ChildLayer”. - In Sketch, all artboards are imported, but only the first one will be visible by default. - In Sketch, make sure you have the elements you like to import don't have negative coordinates as they'll be hidden within Framer. --- #### Compatibility:
Application Compatibility
Sketch 3 Compatible, not recommended
Sketch 3.1 Beta Fully compatible
Photoshop CS6 Fully compatible
Photoshop CC Fully compatible
Photoshop CC 2014 Fully compatible
