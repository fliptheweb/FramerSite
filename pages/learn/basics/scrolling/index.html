{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Scrolling You can scroll a layer contained within a superLayer, as long as its height or width exceeds that of its parent. 
# The superLayer
layerA = new Layer 
	height: 100

# The subLayer
layerB = new Layer 
	height: 200, superLayer: layerA

# Enable scrolling on the superLayer
layerA.scroll = true
Alternatively, you can enable scrolling for a specific direction only: 
layerA.scrollHorizontal = true
layerA.scrollVertical = true
There is one scrolling event, which you can also use to retreive the scrolling positions of a layer. 
# Scroll Event
layerA.on Events.Scroll, ->
	print layerA.scrollX
	print layerA.scrollY
