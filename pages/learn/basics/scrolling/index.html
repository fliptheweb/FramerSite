{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Scrolling A layer can be made scrollable as long as it contains a subLayer that exceeds its height or width. 
# The superLayer
layerA = new Layer 
	height: 100

# The subLayer
layerB = new Layer 
	height: 200, superLayer: layerA

# Enable scrolling on the superLayer
layerA.scroll = true
A layer can be made only horizontally or vertically scrollable. 
layerA.scrollHorizontal = true
layerA.scrollVertical = true
Framer also contains a scrolling event. Within the scroll event, the positions of the scrollable layer can be retrieved. Layer animations can be triggered based on these positions with comparators. 
# Scroll Event
layerA.on Events.Scroll, ->
	print layerA.scrollX
	print layerA.scrollY
Summary
  • Layers containing larger subLayers can be made scrollable
  • You can define a specific scroll direction
  • Layers animations can be triggered on scroll
