Getting this to feel just right involves a lot of physics. The `ScrollComponent` in Framer takes care of this, while remaining completely customisable. A ScrollComponent is built with two layers. First, the ScrollComponent itself, which serves as a masking layer. Secondly, it contains a content layer that has draggable enabled and pre-defined constraints. The size of the content layer is calculated for you, based on the size of its subLayers. 
# Create a ScrollComponent
scroll = new ScrollComponent 
	width:120, height:120

# The content layer
layerA = new Layer 
	width:120, height:200
	superLayer: scroll.content

# Disable horizontal scrolling
scroll.scrollHorizontal = false
A great way to bring your static designs to life with Framer is by using the [Import](/learn/import) feature. To add scrolling behaviour to imported layers, you can wrap them in a ScrollComponent, using `ScrollComponent.wrap()`. --- ##### Content Inset Secondly, it contains a content layer that has draggable enabled and pre-defined constraints. The size of the content layer is calculated for you, based on the size of its subLayers. 
# Create a ScrollComponent
scroll = new ScrollComponent 
	width:120, height:120

# Define the contentInset
scroll.contentInset = {
	top: 40
	bottom: 40 
	right: 0
	left: 0 	
}
Just like a draggable layer, ScrollComponents can be made only horizontally or vertically scrollable. 
scroll.scrollHorizontal = true
scroll.scrollVertical = true
##### Events Framer also contains a scrolling event. Within the scroll event, the positions of the scrollable layer can be retrieved. Layer animations can be triggered based on these positions with conditionals. 
# Create a ScrollComponent
scroll = new ScrollComponent 
	width:120, height:120

# Check the vertical scrolling distance
scroll.on Events.Scroll, ->

	# When below -10, animate a layer
	if scroll.scrollY < -10
		layerA.animate 
			properties: { scale: 1 }
			curve: "spring(400,30,0)"
Summary
  • Layers containing larger subLayers can be made scrollable
  • You can define a specific scroll direction
  • Layers animations can be triggered on scroll
