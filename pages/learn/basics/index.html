{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Layers A layer is a rectangle that draws elements. Layers can display images, videos or text, and contain a set of basic properties. Every layer inherits a few default properties: 
# Create a layer 
layerA = new Layer 
	x:0, y:0, width:100, height:100
##### Hierarchy Parent layers are called superLayers, and child layers are called subLayers. Layers inherit properties from their superLayers. You can define a layers superLayer as a property, too. 
# Two ways to define hierarchy
layerB.superLayer = layerA
layerA.subLayer = layerB
##### Positioning You can **center** a layer within its superLayer by calling the center() function: 
layerA.center()
You can also dynamically retreive values of a layers X and Y position, as well as its **minX** and **maxX** position. 
# Properties of layerA:
# x:40, y:40, width:80, height:80

layerB.x = layerA.X     # 40
layerB.x = layerA.minX  # 40
layerB.x = layerA.maxX  # 120
##### Properties Layers can have a range of properties. Besides being able to define an image, background, or video, you can also transform, hide layers, scale layers and much more. 
layerB = new Layer
	backgroundColor:"#28AFFA",
	width:60, height:60,
	scale:1, borderRadius:"3px"
You can also define and override properties after having defined a layer. Note the different notation. 
layerA.rotation = 45,
layerA.opacity = 0.5
layerA.scale = 0.5
##### Layer types Layers can contain images, videos and text. 
# imageLayer
layerA = new Layer
	image: "image.png"

# videoLayer
layerA = new Layer
	video: "video.mp4"

# Labeled Layer
layerA = new Layer
layerA.html = "Label"
