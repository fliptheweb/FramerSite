{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Animation Most properties of a layer can be animated, including x, y, width, height, scale, opacity, rotation and more. Multiple properties can be animated at once. 
layerA.animate
	properties:
		opacity: 0.5
##### Timing Except when using spring curves, you can define the duration of an animation. Animations can also be delayed and repeated. Animation time and delay are defined in seconds. 
layerA.animate
	properties:
		opacity: 0.5
	curve: "ease"
	repeat: 1
	delay: 2
    time: 1
##### Curves Animations optionally take a curve that describe the type of animation. You can use pre-defined curves like 'linear' or 'ease-in', custom bezier curves and spring animations. Below is an overview of different curves. 
# Linear curve
curve:"linear"

# Easing curve
curve:"ease-in-out"

# Easing curve
curve:"spring(200,15,0)"

Summary

  • Multiple properties can be animated at once
  • Animations can be timed, delayed and repeated
  • Framer contains various animation curves, including spring curves
