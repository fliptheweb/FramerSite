{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### Dragging You can create a draggable layer by enabling it: 
# Make the layer draggable 
layerA.draggable.enabled = true
You can also specify the dragging resistance. Setting the resistance to 0 will disable dragging entirely. 
# Set dragging resistance
layerA.draggable.speedX = 1
layerA.draggable.speedY = 0.5
There are three dragging events in Framer: 
# Drag Events
layerA.on Events.DragStart, -> 
    ...
layerA.on Events.DragMove, -> 
    ...
layerA.on Events.DragEnd, -> 
    ...
