{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Dragging You can create a draggable layer by setting draggable.enabled to true. You can also specify the dragging resistance. This is especially useful if you'd like an element to be draggable only horizontally or vertically: simply set either the speedX or speedY to 0. 
# Make the layer draggable 
layerA.draggable.enabled = true

# Set dragging resistance
layerA.draggable.speedX = 1
layerA.draggable.speedY = 0
There are three dragging events in Framer: 
# Drag Events
layerA.on Events.DragStart, -> 
    ...
layerA.on Events.DragMove, -> 
    ...
layerA.on Events.DragEnd, -> 
    ...
