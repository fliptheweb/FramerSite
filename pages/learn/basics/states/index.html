{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### States Think of a state as a set of properties. A layer can have multiple states. You can add, remove or override states, as well as animate between them. 
# Add states
layerA.states.add 
    second: {scale: 0.5} 
    third:  {rotation:45} 
    
# Set the animation options
layerA.states.animationOptions = 
    curve: "spring(600,30,0)"

# Toggle states on click
layerA.on Events.Click, -> 
    imageLayer.states.next()
