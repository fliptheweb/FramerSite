About

Learn more about the origins of Framer and the team behind it.

{% endblock pageheader %} {% block body %}
Framer started as an open source prototyping framework, written in JavaScript.

In June 2014, we released Framer Studio, a prototyping application for Mac, built on Framer.js, complete with its own code editor, instant visual feedback, Photoshop & Sketch integration and much more.

We’re creating a prototyping tool to help designers create better products.

We’re excited about the future of interaction design tools. We believe prototyping encourages exploration, helps designers effectively communicate concepts and helps engineers implement them.
