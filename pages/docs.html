### Animations Framer supports many different types of animations, including springs and bezier curves.
Property Description Examples
layer layer that is animated
properties properties being animated (x, y, width, height, opacity, scale, rotation, filters)
curve curve used for animation:
  • "linear"
  • "bezier-curve"
  • "spring-rk4" Runge-Kutta spring
  • "spring-dho" Damping harmonic oscillator spring
  • "spring" same as "spring-rk4"
curveOptions options for used curve
  • bezier-curve: "ease-in", "ease-out", "ease-in-out", [0, 1, 0, 1]
  • spring-rk4: friction, tension, velocity, tolerance
  • spring-dho: stiffness, damping, mass, velocity, tolerance
 1
time duration of the animation in seconds (ignored for springs)
delay amount of seconds to delay the animation before start
repeat number of times to repeat animation
start() start the animation
stop() stop the animation
reverse() returns the opposite animation to this one
on(event, function) listen to events: "start", "stop", "end"

### Utilities Utilities is a collection of functions that make things easier to do. Lots of handy stuff here. Be sure to check the source to see everything.
Function Description Examples
cycle([1, 2, ...]) Returns a function that returns the next value every time you call it.
randomColor(alpha) Returns a random color string.
delay(time, function) Executes the function after time in seconds
interval(time, function) Executes the function every x seconds
throttle(time, function) Call function at most once every x seconds
debounce(time, function) Call function only when being stopped calling for x seconds
convertPoint(point, layerA, layerB) Converts a point location from one layers coordinate system to another.
labelLayer(layer, "text") Add centered text label to a layer.
uuid() Returns a random hex uuid string.
round(number, decimals) Rounds a number to x decimals.
isWebKit() Returns wether this is a webkit browser.
isMobile() Returns wether this is a mobile device.
isTouch() Returns wether this is a touch device.
