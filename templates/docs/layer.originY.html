###### layer.originY number Sets the y origin for scale, rotate and skew tranformations. So for example point that a layer rotates around. The origin is defined as a number between 0 and 1 where 0 is the most top edge of the layer and 1 the most bottom edge. The default value is 0.5, the center of the layer. 
layerA = new Layer()
layerA.rotation = 45
layerA.originY = 0
layerA.originY = 1
