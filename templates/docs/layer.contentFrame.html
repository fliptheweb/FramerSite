###### layer.contentFrame()

Returns: Object

The calculated frame for the total size of all the subLayers combined. 
layerA = new Layer()
layerB = new Layer({x:0, width:100, superLayer:layerA})
layerC = new Layer({x:100, width:300, superLayer:layerA})

print layerA.contentFrame()
# Output: {x:0, y:0, width:400, height:100}
