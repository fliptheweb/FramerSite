The Canvas object contains the size for the current entire document in pixels. It will change if you resize your document by resizing the window it is in. ###### Canvas.width number The width of the current entire document in pixels. This property is read-only. 
print Canvas.width
# Output: 640
###### Canvas.height number The height of the current entire document in pixels. This property is read-only. 
print Canvas.height
# Output: 480
###### Canvas.size object The width and height of the current entire document in pixels. This property is read-only. 
print Canvas.size
# Output: {width:640, height:480}
