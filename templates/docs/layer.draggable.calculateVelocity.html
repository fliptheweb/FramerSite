###### layer.draggable.calculateVelocity()

Returns: Object

The averaged calculated dragging velocity in pixels per second in the last 0.1 second. The positive/negative values will indicate the direction of the drag. 
layerA = new Layer()
layerA.draggable.enabled = true

# Now do some dragging
print layer.draggable.calculateVelocity()
# Output: {x:0.234, y:-38.4}

# Just a bit later...
print layer.draggable.calculateVelocity()
# Output: {x:60.3, y:0}
