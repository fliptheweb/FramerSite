###### layer.ignoreEvents boolean Sets wether the layer responds to any user events like click or touch. When disabled, no user events on the layer will be emitted. The default value for this is true because performance is better when layers don't have to account for events. Framer automatically disables it when you add an event listener. 
layerA = new Layer()

layerA.on Events.Click, ->
	print "Click!"

# Now it won't respond to a click
layerA.ignoreEvents = true

# Now it will
layerA.ignoreEvents = false
