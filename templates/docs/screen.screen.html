The Screen object contains the size for the current device screen. The size will change when you update to a different device. If the device is full screen, it will be equal to the Canvas size. ###### Screen.width number The width of the current device screen in pixels. This property is read-only. 
print Screen.width
# Output: 640
###### Screen.height number The height of the current device screen in pixels. This property is read-only. 
print Screen.height
# Output: 1080
###### Screen.size object The width and height of the current device screen in pixels. This property is read-only. 
print Screen.size
# Output: {width:640, height:1080}
