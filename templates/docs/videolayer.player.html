###### videolayer.player string Player object for the video layer. Allows you to control the video and listen for playback events. For a full reference see this overview. For the events see this overview. 
# Set on creation
layerA = new VideoLayer({video: "hello.mp4"})

# Enable auto play
layerA.player.autoplay = true

# Play the video
layerA.player.play()

# Jump to 5 seconds
layerA.player.fastSeek(5)

# Listen to paused playback
Events.wrap(layerA.player).on "pause", -> print "Video paused"
