###### layer.draggable.speedX number Modify the horizontal dragging speed. The value is in pixels per mouse moved pixels. The default value is 1. When set lower then 1 dragging will be slower than mouse movement and vice versa. You can set the value to 0 to disable horizontal dragging. 
layerA = new Layer()
layerA.draggable.enabled = true

# Make horizontal dragging slow
layerA.draggable.speedX = 0.1

# Make horizontal dragging fast
layerA.draggable.speedX = 10
