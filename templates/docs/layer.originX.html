###### layer.originX number Sets the x origin for scale, rotate and skew tranformations. So for example point that a layer rotates around. The origin is defined as a number between 0 and 1 where 0 is the most left edge of the layer and 1 the most right edge. The default value is 0.5, the center of the layer. 
layerA = new Layer()
layerA.rotation = 45
layerA.originX = 0
layerA.originX = 1
