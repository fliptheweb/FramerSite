###### ScrollComponent.scrollToClosestLayer(originXnumber, originYnumber) Scroll to the closest content layer, using the given origin. The default values are (0,0) which is the top-left corner, (0.5, 0.5) the center and (1,1) the bottom-right corner. 
scroll = new ScrollComponent()
layerA = new Layer({superLayer:scroll.content})

# Scroll to the center of layerA
scroll.scrollToClosestLayer(0.5, 0.5)
