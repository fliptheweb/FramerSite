###### layer.on(eventName string, handler function) Start listening for an event on this layer. For a full list of events see this overview. When an event is called the first argument is the event information. Depending on the specific event this can contain mouse positions, mouse deltas etc. The second argument is always the layer that the event occurred to. 
layerA = new Layer()
layerA.name = "Layer A"

layerA.on Events.Click, (event, layer) ->
    print "This layer was clicked", layer.name

# Output: "This layer was clicked", "Layer A"
