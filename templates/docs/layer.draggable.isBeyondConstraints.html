###### layer.draggable.isBeyondConstraints boolean See if the draggable layer is currently beyond its dragging constraints.
This property is read-only. 
layerA = new Layer()
layerA.draggable.enabled = true

# Set dragging constraints
layerA.draggable.constraints = {x:100, y:100, width:400, height:400}

# On move, see if the layer is beyond constraints or not
layerA.on Events.Move, ->
	print layerA.draggable.isBeyondConstraints
