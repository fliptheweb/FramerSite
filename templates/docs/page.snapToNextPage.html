###### page.snapToNextPage(direction string, animate boolean, animationOptions ) Snap to a specific the next page. Takes three arguments: direction, animate (true or false) and animationOptions. By default, the direction is set to "right" and animate is to true. 
page = new PageComponent({
	width: Screen.width, height: Screen.height
})

pageOne = new Layer({
	width: page.width, height: page.height
	superLayer: page.content
	backgroundColor: "#28affa"
})

pageTwo = new Layer({
	width: page.width, height: page.height
	backgroundColor: "#90D7FF"
})

page.addPage(pageTwo, "right")

# Automatically scroll to pageTwo
page.snapToPage(pageTwo)
In the example above, we can customize the scrolling animation by defining custom animationOptions as well. 
# Define a slower easing curve
page.snapToPage(pageTwo, true, animationOptions={curve:"ease", time:2})
