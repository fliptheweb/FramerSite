###### layer.screenFrame object Allows you to set or capture the absolute position of this layer on the screen, ignoring the inherited coordinate system from it's parents. Or, the exact position on the screen. 
layerA = new Layer({x:100})
layerB = new Layer({x:100, superLayer:layerA})

print layerB.screenFrame
# Output: {x:200, y:0, width:100, height:100}

layerB.screenFrame = {x:400, y:0, width:100, height:100}

print layerB.x
# Output: 300
