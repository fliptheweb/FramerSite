###### layer.centerX(offset number) Center this layer horizontally in its superlayer. If there is no superlayer it will be centered relative to the screen. The offset is a pixel offset from the center and optional. 
layerA = new Layer({width:500, height:500})
layerB = new Layer({width:100, height:100, superLayer:layerA})

layerB.centerX()
print layerB.x, layerB.y
# Output: 200, 0

layerB.centerX(20)
print layerB.x, layerB.y
# Output: 220, 0
