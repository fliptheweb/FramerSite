Compatibility

Since Framer is a prototyping toolkit, extensive browser compability is not a priority. Our main goal is achieving optimal performance, to get high framerates and smooth animations.

Framer Compatibility
Framer.js
Browsers Webkit (Chrome, Safari)
Platforms Mac OS X, Windows, Linux
Mobile Platforms iOS, Android, Windows Phone
Generator.app
Imports from Photoshop (13.0.0+), Sketch (3.0.3+)
Platforms Mac OS X, Windows (coming soon)
Framer Studio.app
Imports from Photoshop (13.0.0+), Sketch (3.0.3+)
Platforms Mac OS X 10.8+

Debugging

Because Framer runs in the browser you can use all the debugging tools, like the console, timeline, profiler and debugger. Framer will alert you when a Javascript error occurred.

Key Description
Escape (toggle) Switch between normal & debug view of all layers. Useful to inspect structure and positions.

Questions

There is an active and helpful Framer Community on Facebook.

  • Framer on Windows
    • Yes. The core of Framer is a Javascript library that works on all platforms. We're working on Photoshop compatibility.
  • Framer Studio Volume Licensing
    • For 50 licenses or less you can go directly through our online store. If you need more than 50 copies and have special requests please contact us. You can find the Framer end user license agreement here.
  • Framer Studio Support
    • Framer comes with personal support for technical application issues (crashes, etc.). If you need help learning Framer you can ask questions in our group, where we and other users try to help each other out.
